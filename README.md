## Install

```bash
yarn add @vuepress/theme-modern -D
# OR npm install @vuepress/theme-modern -D
```


## Usage

```js
// .vuepress/config.js
module.exports = {
  theme: '@vuepress/modern',
  themeConfig: {
    // Please head documentation to see the available options.
  }
}
```
